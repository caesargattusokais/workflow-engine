MVP Scope

  • Designer: drag-drop nodes, connect edges, edit properties, export YAML
  • Monitor: instance list, flow diagram with active-node highlight, task panel
  • Server: REST API wrapping workflow-engine-core
  • In-memory storage (DB later via SPI)

Out of MVP Scope

  • Authentication / multi-tenant
  • Collaborative editing
  • Version history UI
  • Production deployment
